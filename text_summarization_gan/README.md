# Generative AI-Based Text Summarization using Advanced GAN Models

This project implements a complete end-to-end framework for text summarization utilizing an advanced Generative Adversarial Network (GAN) architecture. The model is capable of generating high-quality abstractive summaries from both direct text input and various document types (PDF, DOCX, TXT).

## Features
- **Encoder-Decoder Generator**: Transformer-based (T5) generator creating abstractive summaries.
- **Transformer Discriminator**: Determines if a summary is generated (Real vs Fake).
- **Adversarial Training**: Custom training script to simulate SeqGAN/Adversarial Text-GAN training loops.
- **Document Extractors**: Processes PDF, DOCX, and TXT files.
- **Interactive UI**: Beginner-friendly and easy-to-use Streamlit dashboard.
- **GPU Acceleration**: Utilizes PyTorch CUDA.

## Setup Instructions

### 1. Prerequisites
Ensure you have Python 3.8+ installed on your system. A GPU (CUDA-compatible) is highly recommended for faster inference and training.

### 2. Install Dependencies
Open your terminal (in VS Code or Command Prompt), navigate to this project's root folder, and run:
```bash
pip install -r requirements.txt
```
*Note: Make sure to install the PyTorch version that perfectly matches your CUDA system if you plan to utilize GPU acceleration.*

### 3. Running the Streamlit Application
Start the frontend interface:
```bash
streamlit run app.py
```
This will open the user interface locally in your web browser, where you can paste text or upload documents to get a summarized text generated by the Generator.

### 4. Training the GAN
To train the customized adversarial text GAN, ensure you are in the project's root folder, and execute:
```bash
python train.py
```
This script will initialize both the `TextGenerator` and `TextDiscriminator`. It uses continuous adversarial approximation over a mock CNN/DailyMail dataset layout structure to simulate an advanced SeqGAN environment. The trained models will be evaluated and exported to the `models/saved/` directory.
